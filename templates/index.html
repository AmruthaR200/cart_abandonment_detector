Objective: To develop a machine learning-based web application that predicts the likelihood of a user abandoning their shopping cart in an e-commerce platform.

Problem Statement: Shopping cart abandonment is a major issue in e-commerce. Our goal is to detect this behavior early using user interaction data, enabling businesses to take proactive actions like sending reminders or offering discounts.

Solution: The system takes user behavior input like time on site, pages viewed, and cart value, and uses a trained ML model to predict whether the cart will be abandoned.

Modules:

Frontend: Dark-themed UI with form and smooth navigation

Backend: Flask app that handles requests and serves predictions

ML Model: Classification algorithm trained to detect abandonment patterns

Deployment: Can be hosted locally or on platforms like Heroku

Tech Stack: HTML, CSS, Flask, Python, Scikit-learn, Jupyter Notebook

Outcome: A practical solution that helps improve conversion rates by predicting user intentions in real-time.
